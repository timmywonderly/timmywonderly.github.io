Chapter 11

Briefly describe the four gates explained in chapter 11. Describe the behavior of each gate based on the input values to the gate.

OR Gate= output is 1 if any inputs are 1, if else 0. AND Gate= output is 1 if all inputs are 1, if else 0. NOR Gate= output is 1 if no inputs are 1, if else 0. NAND Gate= output is 1 unless all inputs are 1.
